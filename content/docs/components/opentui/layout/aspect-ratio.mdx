## Usage

```tsx
import { AspectRatio } from "@/components/ui/aspect-ratio";
```

```tsx
<AspectRatio ratio={16 / 9} width={60}>
  <Text>Widescreen content</Text>
</AspectRatio>
```

## API Reference

### AspectRatio

| Prop       | Type        | Default    |
| ---------- | ----------- | ---------- |
| `children` | `ReactNode` | `required` |
| `ratio`    | `number`    | `16 / 9`   |
| `width`    | `number`    | `80`       |

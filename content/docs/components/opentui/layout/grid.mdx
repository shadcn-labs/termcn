## Usage

```tsx
import { Grid } from "@/components/ui/grid";
```

```tsx
<Grid columns={3} gap={1}>
  <Text>A</Text>
  <Text>B</Text>
  <Text>C</Text>
  <Text>D</Text>
  <Text>E</Text>
  <Text>F</Text>
</Grid>
```

## API Reference

### Grid

| Prop       | Type        | Default    |
| ---------- | ----------- | ---------- |
| `columns`  | `number`    | `required` |
| `gap`      | `number`    | `0`        |
| `children` | `ReactNode` | `required` |

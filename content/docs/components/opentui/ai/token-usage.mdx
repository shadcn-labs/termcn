## Usage

```tsx
import { TokenUsage, ContextMeter } from "@/components/ui/token-usage";
```

```tsx
<TokenUsage prompt={2450} completion={890} model="gpt-4o" showCost />
```

## API Reference

### TokenUsage

| Prop         | Type      | Default    |
| ------------ | --------- | ---------- |
| `prompt`     | `number`  | `required` |
| `completion` | `number`  | `required` |
| `model`      | `string`  | -          |
| `showCost`   | `boolean` | `false`    |

### ContextMeter

| Prop          | Type      | Default    |
| ------------- | --------- | ---------- |
| `used`        | `number`  | `required` |
| `limit`       | `number`  | `required` |
| `label`       | `string`  | -          |
| `showPercent` | `boolean` | `true`     |
| `warnAt`      | `number`  | `75`       |
| `criticalAt`  | `number`  | `90`       |
| `width`       | `number`  | `20`       |

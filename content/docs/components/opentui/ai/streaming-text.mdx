## API Reference

### StreamingText

| Prop          | Type                         | Default |
| ------------- | ---------------------------- | ------- |
| `text`        | `string`                     | -       |
| `stream`      | `AsyncIterable<string>`      | -       |
| `cursor`      | `boolean`                    | `true`  |
| `animate`     | `boolean`                    | `false` |
| `speed`       | `number`                     | `30`    |
| `onComplete`  | `(fullText: string) => void` | -       |
| `cursorColor` | `string`                     | -       |

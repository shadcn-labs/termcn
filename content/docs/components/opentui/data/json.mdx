## Usage

```tsx
import { JSONView } from "@/components/ui/json";
```

```tsx
<JSONView
  label="package.json"
  data={{
    name: "my-cli",
    version: "1.0.0",
    dependencies: { ink: "^5.0.0" },
  }}
/>
```

## API Reference

### JSONView

| Prop        | Type      | Default    |
| ----------- | --------- | ---------- |
| `data`      | `unknown` | `required` |
| `indent`    | `number`  | `2`        |
| `collapsed` | `boolean` | `false`    |
| `label`     | `string`  | -          |

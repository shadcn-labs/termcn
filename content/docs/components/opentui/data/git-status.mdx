## API Reference

### GitStatus

| Prop       | Type     | Default    |
| ---------- | -------- | ---------- |
| `branch`   | `string` | `required` |
| `staged`   | `number` | `0`        |
| `modified` | `number` | `0`        |
| `ahead`    | `number` | `0`        |
| `behind`   | `number` | `0`        |

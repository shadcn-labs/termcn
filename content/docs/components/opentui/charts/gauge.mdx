## Usage

```tsx

```

```tsx
<Gauge value={72} max={100} label="CPU Usage" size="md" />
```

## API Reference

### Gauge

| Prop    | Type                   | Default    |
| ------- | ---------------------- | ---------- |
| `value` | `number`               | `required` |
| `min`   | `number`               | `0`        |
| `max`   | `number`               | `100`      |
| `label` | `string`               | -          |
| `color` | `string`               | -          |
| `size`  | `"sm" \| "md" \| "lg"` | `"md"`     |

## Usage

```tsx
import { LineChart } from "@/components/ui/line-chart";
```

```tsx
<LineChart
  data={[
    { label: "Jan", value: 30 },
    { label: "Feb", value: 45 },
    { label: "Mar", value: 28 },
  ]}
  width={40}
  height={10}
  title="Monthly Sales"
/>
```

## API Reference

### LineChart

| Prop       | Type                                              | Default    |
| ---------- | ------------------------------------------------- | ---------- |
| `data`     | `(number \| { label?: string; value: number })[]` | `required` |
| `width`    | `number`                                          | `40`       |
| `height`   | `number`                                          | `10`       |
| `title`    | `string`                                          | -          |
| `color`    | `string`                                          | -          |
| `showAxes` | `boolean`                                         | `true`     |

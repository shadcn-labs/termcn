## Usage

```tsx
import { PieChart } from "@/components/ui/pie-chart";
```

```tsx
<PieChart
  data={[
    { label: "TypeScript", value: 45 },
    { label: "Rust", value: 25 },
    { label: "Go", value: 20 },
  ]}
  radius={6}
/>
```

## API Reference

### PieChart

| Prop              | Type             | Default    |
| ----------------- | ---------------- | ---------- |
| `data`            | `PieChartItem[]` | `required` |
| `radius`          | `number`         | `5`        |
| `showLegend`      | `boolean`        | `true`     |
| `showPercentages` | `boolean`        | `true`     |

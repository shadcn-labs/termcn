## Usage

```tsx
import { Popover } from "@/components/ui/popover";
```

```tsx
<Popover
  isOpen={true}
  title="Info"
  trigger={<Text bold>[?] Help</Text>}
  onClose={() => close()}
>
  <Text>Press Enter to confirm, Esc to cancel.</Text>
</Popover>
```

## API Reference

### Popover

| Prop       | Type         | Default    |
| ---------- | ------------ | ---------- |
| `trigger`  | `ReactNode`  | `required` |
| `children` | `ReactNode`  | `required` |
| `isOpen`   | `boolean`    | `false`    |
| `onClose`  | `() => void` | -          |
| `title`    | `string`     | -          |

## API Reference

### ProgressBar

| Prop          | Type      | Default    |
| ------------- | --------- | ---------- |
| `value`       | `number`  | `required` |
| `total`       | `number`  | -          |
| `width`       | `number`  | `30`       |
| `showPercent` | `boolean` | `true`     |
| `showEta`     | `boolean` | `false`    |
| `fillChar`    | `string`  | `"█"`      |
| `emptyChar`   | `string`  | `"░"`      |
| `color`       | `string`  | -          |
| `label`       | `string`  | -          |

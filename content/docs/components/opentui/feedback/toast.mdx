## Usage

```tsx
import { Toast } from "@/components/ui/toast";
```

```tsx
<Toast
  variant="success"
  message="Settings saved successfully."
  duration={5000}
  onDismiss={() => console.log("toast closed")}
/>
```

## API Reference

### Toast

| Prop        | Type                                          | Default    |
| ----------- | --------------------------------------------- | ---------- |
| `message`   | `string`                                      | `required` |
| `variant`   | `"success" \| "error" \| "warning" \| "info"` | `"info"`   |
| `duration`  | `number`                                      | `3000`     |
| `onDismiss` | `() => void`                                  | -          |
| `icon`      | `string`                                      | -          |

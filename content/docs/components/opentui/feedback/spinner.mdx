## API Reference

### Spinner

| Prop     | Type                                                                              | Default  |
| -------- | --------------------------------------------------------------------------------- | -------- |
| `type`   | `SpinnerName` from [`cli-spinners`](https://github.com/sindresorhus/cli-spinners) | `"dots"` |
| `label`  | `string`                                                                          | -        |
| `color`  | `string`                                                                          | -        |
| `fps`    | `number` — used only with custom `frames`                                         | `12`     |
| `frames` | `string[]`                                                                        | -        |

```

</Alert>
```

## API Reference

### Alert

| Prop          | Type                                                                                         | Default  |
| ------------- | -------------------------------------------------------------------------------------------- | -------- |
| `variant`     | `"success" \| "error" \| "warning" \| "info"`                                                | `"info"` |
| `title`       | `string`                                                                                     | -        |
| `children`    | `ReactNode`                                                                                  | -        |
| `icon`        | `string`                                                                                     | -        |
| `bordered`    | `boolean`                                                                                    | `true`   |
| `borderStyle` | `"single" \| "double" \| "round" \| "bold" \| "singleDouble" \| "doubleSingle" \| "classic"` | -        |
| `color`       | `string`                                                                                     | -        |
| `paddingX`    | `number`                                                                                     | `1`      |
| `paddingY`    | `number`                                                                                     | `0`      |

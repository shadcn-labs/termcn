## Usage

```tsx

```

```tsx
<Banner variant="info" title="Update">
  A new version is available. Run npx update to upgrade.
</Banner>
```

## API Reference

### Banner

| Prop          | Type                                                       | Default    |
| ------------- | ---------------------------------------------------------- | ---------- |
| `children`    | `ReactNode`                                                | `required` |
| `variant`     | `"info" \| "warning" \| "error" \| "success" \| "neutral"` | `"info"`   |
| `icon`        | `string`                                                   | -          |
| `title`       | `string`                                                   | -          |
| `dismissible` | `boolean`                                                  | `false`    |
| `onDismiss`   | `() => void`                                               | -          |
| `color`       | `string`                                                   | -          |
| `accentChar`  | `string`                                                   | `"┃"`      |
| `gap`         | `number`                                                   | `1`        |

## API Reference

### TagInput

| Prop          | Type                       | Default                   |
| ------------- | -------------------------- | ------------------------- |
| `value`       | `string[]`                 | -                         |
| `onChange`    | `(tags: string[]) => void` | -                         |
| `placeholder` | `string`                   | `"Type and press Enter…"` |
| `maxTags`     | `number`                   | -                         |
| `delimiter`   | `string`                   | -                         |

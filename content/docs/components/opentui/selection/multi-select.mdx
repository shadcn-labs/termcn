## Usage

```tsx
import { MultiSelect } from "@/components/ui/multi-select";
```

```tsx
<MultiSelect
  options={[
    { value: "react", label: "React" },
    { value: "vue", label: "Vue" },
    { value: "svelte", label: "Svelte" },
  ]}
  onChange={(values) => {}}
/>
```

## API Reference

### MultiSelect

| Prop        | Type                     | Default    |
| ----------- | ------------------------ | ---------- |
| `options`   | `MultiSelectOption<T>[]` | `required` |
| `value`     | `T[]`                    | -          |
| `onChange`  | `(values: T[]) => void`  | -          |
| `onSubmit`  | `(values: T[]) => void`  | -          |
| `cursor`    | `string`                 | `"›"`      |
| `checkmark` | `string`                 | `"◉"`      |
| `height`    | `number`                 | -          |

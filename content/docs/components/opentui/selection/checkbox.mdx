## API Reference

### Checkbox

| Prop                | Type                         | Default |
| ------------------- | ---------------------------- | ------- |
| `checked`           | `boolean`                    | -       |
| `onChange`          | `(checked: boolean) => void` | -       |
| `label`             | `string`                     | -       |
| `indeterminate`     | `boolean`                    | `false` |
| `disabled`          | `boolean`                    | `false` |
| `id`                | `string`                     | -       |
| `checkedIcon`       | `string`                     | `"■"`   |
| `uncheckedIcon`     | `string`                     | `"□"`   |
| `indeterminateIcon` | `string`                     | `"▪"`   |

## API Reference

### Badge

| Prop          | Type                                                                                         | Default     |
| ------------- | -------------------------------------------------------------------------------------------- | ----------- |
| `children`    | `string`                                                                                     | `required`  |
| `variant`     | `"default" \| "success" \| "warning" \| "error" \| "info" \| "secondary"`                    | `"default"` |
| `color`       | `string`                                                                                     | -           |
| `bold`        | `boolean`                                                                                    | `false`     |
| `bordered`    | `boolean`                                                                                    | `true`      |
| `borderStyle` | `"single" \| "double" \| "round" \| "bold" \| "singleDouble" \| "doubleSingle" \| "classic"` | `"round"`   |
| `paddingX`    | `number`                                                                                     | `1`         |

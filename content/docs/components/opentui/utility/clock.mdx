## Usage

```tsx
import { Clock } from "@/components/ui/clock";
```

```tsx
<Clock format="12h" showSeconds showDate timezone="America/New_York" />
```

## API Reference

### Clock

| Prop          | Type             | Default |
| ------------- | ---------------- | ------- |
| `format`      | `"12h" \| "24h"` | `"24h"` |
| `showSeconds` | `boolean`        | `true`  |
| `showDate`    | `boolean`        | `false` |
| `timezone`    | `string`         | -       |
| `color`       | `string`         | -       |
| `size`        | `"sm" \| "lg"`   | `"sm"`  |

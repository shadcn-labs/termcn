## API Reference

### ErrorBoundary

| Prop       | Type                                                       | Default    |
| ---------- | ---------------------------------------------------------- | ---------- |
| `children` | `ReactNode`                                                | `required` |
| `fallback` | `ReactNode`                                                | -          |
| `onError`  | `(error: Error, info: { componentStack: string }) => void` | -          |
| `title`    | `string`                                                   | `"Error"`  |

## Usage

```tsx
import { EmbeddedTerminal } from "@/components/ui/embedded-terminal";
```

```tsx
<EmbeddedTerminal
  command="bash"
  args={["-c", "echo Hello world"]}
  width={80}
  height={24}
  onExit={(code) => console.log(`Exited with ${code}`)}
/>
```

## API Reference

### EmbeddedTerminal

| Prop      | Type                     | Default    |
| --------- | ------------------------ | ---------- |
| `command` | `string`                 | `required` |
| `args`    | `string[]`               | `[]`       |
| `cwd`     | `string`                 | -          |
| `width`   | `number`                 | `80`       |
| `height`  | `number`                 | `24`       |
| `onExit`  | `(code: number) => void` | -          |

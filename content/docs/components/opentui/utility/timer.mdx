## Usage

```tsx
import { Timer } from "@/components/ui/timer";
```

```tsx
<Timer duration={300} format="ms" label="Pomodoro" autoStart />
```

## API Reference

### Timer

| Prop         | Type                   | Default    |
| ------------ | ---------------------- | ---------- |
| `duration`   | `number`               | `required` |
| `onComplete` | `() => void`           | -          |
| `autoStart`  | `boolean`              | `false`    |
| `format`     | `"hms" \| "ms" \| "s"` | `"hms"`    |
| `color`      | `string`               | -          |
| `label`      | `string`               | -          |

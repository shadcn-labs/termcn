## API Reference

### Image

| Prop       | Type                                       | Default    |
| ---------- | ------------------------------------------ | ---------- |
| `src`      | `string`                                   | `required` |
| `width`    | `number`                                   | `20`       |
| `height`   | `number`                                   | -          |
| `protocol` | `"auto" \| "iterm2" \| "kitty" \| "ascii"` | `"auto"`   |
| `alt`      | `string`                                   | -          |

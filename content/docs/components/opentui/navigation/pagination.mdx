## Usage

```tsx

```

```tsx
<Pagination total={10} current={page} onChange={(p) => setPage(p)} />
```

## API Reference

### Pagination

| Prop        | Type                     | Default    |
| ----------- | ------------------------ | ---------- |
| `total`     | `number`                 | `required` |
| `current`   | `number`                 | `required` |
| `onChange`  | `(page: number) => void` | -          |
| `showEdges` | `boolean`                | `true`     |
| `siblings`  | `number`                 | `1`        |

## Usage

```tsx
import { BarChart } from "@/components/ui/bar-chart";
```

```tsx
<BarChart
  data={[
    { label: "Mon", value: 12 },
    { label: "Tue", value: 28 },
  ]}
  width={40}
  showValues
  title="Weekly Traffic"
/>
```

## API Reference

### BarChart

| Prop         | Type                         | Default        |
| ------------ | ---------------------------- | -------------- |
| `data`       | `BarChartItem[]`             | `required`     |
| `direction`  | `"horizontal" \| "vertical"` | `"horizontal"` |
| `width`      | `number`                     | `30`           |
| `height`     | `number`                     | `10`           |
| `showValues` | `boolean`                    | `true`         |
| `title`      | `string`                     | -              |

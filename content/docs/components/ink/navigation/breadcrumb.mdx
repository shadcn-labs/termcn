## Usage

```tsx
import { Breadcrumb } from "@/components/ui/breadcrumb";
```

```tsx
<Breadcrumb
  items={[
    { key: "home", label: "Home", onSelect: () => {} },
    { key: "docs", label: "Docs" },
  ]}
  activeKey="docs"
/>
```

## API Reference

### Breadcrumb

| Prop        | Type               | Default    |
| ----------- | ------------------ | ---------- |
| `items`     | `BreadcrumbItem[]` | `required` |
| `separator` | `string`           | `"›"`      |
| `activeKey` | `string`           | -          |

### BreadcrumbItem

| Prop       | Type         | Default    |
| ---------- | ------------ | ---------- |
| `label`    | `string`     | `required` |
| `key`      | `string`     | `required` |
| `onSelect` | `() => void` | -          |

## Usage

```tsx
import { KeyValue } from "@/components/ui/key-value";
```

```tsx
<KeyValue
  items={[
    { key: "Name", value: "my-app" },
    { key: "Version", value: "2.1.0" },
    { key: "License", value: "MIT" },
  ]}
/>
```

## API Reference

### KeyValue

| Prop         | Type             | Default    |
| ------------ | ---------------- | ---------- |
| `items`      | `KeyValueItem[]` | `required` |
| `keyWidth`   | `number`         | -          |
| `separator`  | `string`         | `":"`      |
| `keyColor`   | `string`         | -          |
| `valueColor` | `string`         | -          |

### KeyValueItem

| Prop    | Type        | Default    |
| ------- | ----------- | ---------- |
| `key`   | `string`    | `required` |
| `value` | `ReactNode` | `required` |
| `color` | `string`    | -          |

## Usage

```tsx
import { MaskedInput } from "@/components/ui/masked-input";
```

```tsx
<MaskedInput
  mask="(###) ###-####"
  label="Phone Number"
  placeholder="(555) 123-4567"
/>
```

## API Reference

### MaskedInput

| Prop          | Type                      | Default    |
| ------------- | ------------------------- | ---------- |
| `mask`        | `string`                  | `required` |
| `value`       | `string`                  | -          |
| `onChange`    | `(value: string) => void` | -          |
| `onSubmit`    | `(value: string) => void` | -          |
| `label`       | `string`                  | -          |
| `placeholder` | `string`                  | -          |
| `autoFocus`   | `boolean`                 | `false`    |
| `id`          | `string`                  | -          |
| `width`       | `number`                  | `40`       |

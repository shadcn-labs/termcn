## Usage

```tsx
import { Help } from "@/components/ui/help";
```

```tsx
<Help
  title="Keyboard Shortcuts"
  keymap={{
    q: "Quit",
    h: "Help",
    j: "Move down",
    k: "Move up",
  }}
/>
```

## API Reference

### Help

| Prop      | Type                     | Default    |
| --------- | ------------------------ | ---------- |
| `keymap`  | `Record<string, string>` | `required` |
| `title`   | `string`                 | -          |
| `compact` | `boolean`                | `false`    |

## API Reference

### Markdown

| Prop        | Type      | Default    |
| ----------- | --------- | ---------- |
| `children`  | `string`  | `required` |
| `width`     | `number`  | -          |
| `streaming` | `boolean` | `false`    |
| `cursor`    | `string`  | `"▌"`      |
